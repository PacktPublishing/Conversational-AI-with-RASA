## install rasa
```bash
pip install rasa
```

## get rasa version
```bash
rasa --version
```

## creat sample project
```bash
rasa init
```

## train Rasa project
```bash
rasa train
```

## Setting up the model server and making inferences
```bash
rasa shell
```

Try to type some queries like `hello` and see the response.

Have Fun!
